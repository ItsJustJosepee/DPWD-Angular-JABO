Proyecto Angular

Sobre el proyecto

Este proyecto está desarrollado con Angular e incluye múltiples componentes y prácticas esenciales del framework, tales como Class Binding, Property Binding, Variables de Plantilla, ngIf/Else, ngFor, así como formularios por plantilla, validaciones y formularios reactivos. También incorpora módulos de comunicación entre componentes padre e hijo en distintas variaciones. El objetivo principal es mostrar la estructura modular del proyecto y cómo los componentes interactúan dentro de un entorno moderno utilizando Angular.
